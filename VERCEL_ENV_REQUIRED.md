# Vercel Environment Variables

The current public website is fully static and does not require any environment variables to build or run.

If backend features (Supabase, email, cron) are added later, set these in the Vercel dashboard (no values committed here):

NEXT_PUBLIC_SUPABASE_URL
NEXT_PUBLIC_SUPABASE_ANON_KEY
NEXT_PUBLIC_SITE_URL
SUPABASE_SERVICE_ROLE_KEY
CRON_SECRET
SMTP_HOST
SMTP_PORT
SMTP_SECURE
SMTP_USER
SMTP_PASSWORD
SMTP_FROM
SMTP_REPLY_TO
