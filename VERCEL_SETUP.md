# Vercel Setup Instructions

The Next.js project is at the **repository root**.

## In Vercel Dashboard:

- **Root Directory**: leave empty (or set to `.`)
- **Build Command**: `npm run build` (default for Next.js)
- **Output Directory**: leave empty (Next.js uses `.next` automatically)
- **Install Command**: `npm install` (default)
- Framework Preset: Next.js (auto-detected)

After changing Root Directory, trigger a new deployment.

No environment variables are required for the current public static site.
